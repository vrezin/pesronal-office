# Finance

Личный и семейный финансовый контур.

## Folders

- `personal-budget/` - бюджет, регулярные расходы, cashflow.
- `family-assets/` - активы семьи.
- `obligations/` - долги, кредиты, обязательства.
- `taxes/` - налоги и документы.
- `investments/` - инвестиции и решения по капиталу.
- `big-decisions/` - крупные финансовые решения.

## Rule

Если вход содержит сумму, актив, обязательство, платеж, налог, договор или финансовый риск, агент должен создать финансовую запись или задачу.
