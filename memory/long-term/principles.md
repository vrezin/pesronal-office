# Principles

## Personal Office

- The repository is an operating system, not a dump of notes.
- New intake should become artifacts, tasks, calendar items, facts, or clarification questions.
- Memory supports recall and planning, but source artifacts remain authoritative.

## Agents

- Read narrowly.
- Route before writing.
- Do not invent missing facts.
- Minimize sensitive data.
- Leave a trace of important processing.
