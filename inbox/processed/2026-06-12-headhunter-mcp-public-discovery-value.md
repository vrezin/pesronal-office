# HeadHunter MCP Public Discovery Value

- Date: 2026-06-12
- Source: user discussion about HH MCP without OAuth

## Conclusion

Even without applicant OAuth, the public HeadHunter MCP already removes a large amount of manual work:

- vacancy search and discovery;
- region and filter lookup;
- vacancy detail review;
- employer/company research;
- similar vacancy exploration;
- early filtering before any resume adaptation.

## Practical Value

- Cuts out manual copy/paste of job descriptions into the intake workflow.
- Reduces the most tedious search-and-filter work.
- Keeps the remaining hard part isolated to resume adaptation and response crafting.

## Next Step

- Use the public MCP immediately for vacancy discovery and screening.
- Keep the OAuth/applicant path as a separate optional enhancement, not a dependency for first value.
