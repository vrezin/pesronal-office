# Inbox

Входящий поток сначала попадает сюда.

## Folders

- `raw/` - сырой ввод: переписка, голосовые заметки, сканы, идеи, фрагменты.
- `processed/` - разобранные входы, по которым уже созданы артефакты или задачи.

## Rule

Если агент не уверен, куда маршрутизировать материал, он создает заметку в `processed/needs-clarification-YYYY-MM-DD.md` и явно формулирует вопрос.
