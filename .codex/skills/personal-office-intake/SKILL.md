# Personal Office Intake

Use for any raw intake that may contain decisions, promises, dates, people, money, risks, opportunities, or next steps.

## Start

1. Read `wiki/README.md`.
2. Read `secretaries/routing-map.md`.
3. If needed, read `wiki/playbooks/personal-office-intake.md`.

## Output Contract

- Save raw/processed trace when the input is important.
- Create or update target artifacts.
- Create `tasks/active/` or `tasks/waiting/` when there is an action or dependency.
- Do not invent missing facts.
