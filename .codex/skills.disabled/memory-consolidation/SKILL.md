# Memory Consolidation

## Cadence

- Weekly: consolidate short-term memory, open loops, and current episodic notes.
- Monthly: review semantic topics, entity records, graph edges, and wiki maps.

## Workflow

1. Read `memory/short-term/current-context.md`, `current-week.md`, and `open-loops.md`.
2. Review current-month episodic memory.
3. Move durable facts into `memory/semantic/topics/`.
4. Update `memory/entities/` for recurring people, companies, projects, places, and tools.
5. Update graph nodes/edges through `knowledge-graph-indexing`.
6. Update `memory/retrieval/index.md` if routes changed.
7. Keep source artifacts authoritative.

## Rules

- Summarize; do not duplicate raw notes.
- Remove or archive stale short-term context only when it is captured elsewhere or no longer useful.
- Preserve uncertainty as uncertainty.
