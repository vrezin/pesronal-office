# Personal Office

Приватный операционный контур для управления личными делами, активностями, проектами, компаниями, встречами, задачами и входящим потоком информации.

Главная идея:

> Входящий поток моей жизни должен превращаться в расписание, задачи, решения, документы и память, с которыми могут работать агенты-секретари.

Этот каталог спроектирован как отдельная приватная репа: личный операционный контур, в котором агенты могут принимать входящий поток и превращать его в устойчивые артефакты.

## Structure

- `inbox/` - сырой и разобранный входящий поток.
- `calendar/` - daily/weekly планы, встречи, поездки.
- `tasks/` - активные задачи, ожидания, someday, done.
- `life/` - дом, семья, отдых, здоровье, образ жизни, рутины, долгосрочные цели.
- `finance/` - личный/семейный баланс, деньги, активы, обязательства, налоги, инвестиции, крупные решения.
- `personal-projects/` - личные проекты: Profi.ru, консультации, эксперименты, opportunities.
- `work/` - личная работа, поиск работы, контракты.
- `companies/` - компании и предпринимательские контуры: FinSOK, AI Studio, будущие компании.
- `people/` - контакты, контекст отношений, follow-ups.
- `secretaries/` - правила для агентов-секретарей.

## Operating Rule

Ничего важного не должно оставаться только в чате.

Если вход содержит решение, обещание, дату, деньги, контакт, встречу, риск или возможность, агент должен создать или обновить соответствующий артефакт.

Каждый новый вход проходит через `secretaries/routing-map.md`. Если маршрут не очевиден, агент создает clarification note в `inbox/processed/`, а не угадывает.

## First Secretaries

- Intake secretary: разбирает входящий поток и маршрутизирует.
- Calendar secretary: ведет встречи, планы, напоминания, подготовку.
- Task secretary: превращает договоренности в задачи и follow-ups.
- Finance secretary: ведет деньги, активы, обязательства и крупные решения.
- Project secretary: ведет личные проекты и возможности.
- Company secretary: ведет контуры компаний.
- Archivist: закрывает недели, чистит inbox, делает summaries.
