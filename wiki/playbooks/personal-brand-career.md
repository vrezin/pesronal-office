# Personal Brand Career Playbook

## Source Of Truth

- `personal-projects/personal-brand/workspace/OPERATING_MODEL.md`
- `personal-projects/personal-brand/workspace/job-intake/`
- `personal-projects/personal-brand/workspace/final-cv/`
- `finance/personal-budget/career-relocation-baseline.md`

## Vacancy / JD Intake

1. Save full JD to `job-intake/jd-archive/YYYY-MM-DD-company-role.md`.
2. Save analysis to `job-intake/analyses/YYYY-MM-DD-company-role-analysis.md`.
3. Use `job-intake/jd-analysis-template.md`.
4. Update `job-intake/INDEX.md`.
5. Update `job-intake/COMPANY_NOTES.md` when company/bucket memory changes.

## Analysis Must Include

- verdict;
- best CV;
- match reasons;
- risks;
- relocation/lifestyle/compensation;
- how to defuse risks;
- questions to ask;
- cover letter or screening answer when useful.

## CV Defaults

- AI / GenAI / automation / agents -> AI Transformation CV.
- Startup / first product / founding CTO -> Startup/Product Engineering CV.
- Large engineering org -> Heavy Enterprise CV.
- Digital modernization / transformation -> Digital Transformation CV.
- Stability / governance / SDLC -> Stability & Governance CV.
- Integrator / 1C / P&L / custom delivery -> Business Unit CV.

## HH Gmail Monitor

Use `automation/prompts/hh-gmail-monitor.md` for scheduled mail scans. Classify HH mail as status update, invitation, new vacancy, or noise. Invitations should create/update high-priority tasks and be starred/marked important when the Gmail connector allows it.

## Guardrails

- Do not browse HH by default.
- Do not invent salary, stack, domain, dates, or responsibilities.
- Always consider Novosibirsk family baseline, relocation cost, school logistics, and target income when role format or compensation matters.
