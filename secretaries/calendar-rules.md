# Calendar Rules

## Meeting Note Template

```md
# YYYY-MM-DD - Meeting Title

## Context

## Participants

## Decisions

## Next Steps

## Follow-ups

## Links
```

## Rules

- If a meeting has a decision, create a task or update the relevant project/company artifact.
- If someone promised something, create `tasks/waiting/`.
- If preparation is needed, create an active task before the meeting.
- If the meeting changes a company/project direction, update that company/project folder.
