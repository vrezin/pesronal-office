# Secretaries

Правила для агентов-секретарей.

Секретарь не просто отвечает в чате. Его работа - превращать входящий поток в устойчивые артефакты:

- документы;
- задачи;
- meeting notes;
- financial records;
- follow-ups;
- weekly summaries;
- project/company updates.

## Core Files

- `routing-map.md` - куда класть разные типы входа.
- `intake-rules.md` - как разбирать сырой вход.
- `calendar-rules.md` - как работать со встречами и временем.
- `task-rules.md` - как создавать и обновлять задачи.
- `privacy-rules.md` - что не писать и как обращаться с чувствительной информацией.
