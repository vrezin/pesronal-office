# Calendar

Планы, встречи, поездки и временные обязательства.

## Folders

- `daily/` - дневные планы и brief.
- `weekly/` - недельные обзоры и планирование.
- `meetings/` - заметки по встречам.
- `travel/` - поездки и логистика.

## Rule

Если вход содержит дату, время, участников или обещание встретиться, агент должен создать meeting note или calendar follow-up.
