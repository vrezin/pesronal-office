# Tasks

Единое место для задач и follow-ups.

## Folders

- `active/` - задачи в работе.
- `waiting/` - ожидания от других людей или внешних событий.
- `someday/` - идеи и задачи без текущего обязательства.
- `done/` - завершенные задачи.

## Rule

Если в тексте есть обещание, дедлайн, следующий шаг или зависимость от другого человека, это должно стать задачей или waiting item.
