## Правила использования

- Для планирования работ по дому сначала проверять `repo/`, затем обновлять контрольные артефакты в `control/`.
- Не перемещать и не переименовывать исходные PDF/Excel/Blend без отдельного запроса.
- Если задача касается смет, читать Excel-источники из `repo/`.
- Если задача касается планировки, инженерки или визуальной проверки, использовать PDF/Blender/рендеры из `repo/`.
- Если задача касается текущего execution-плана, фиксировать результат в `control/current-plan.md`.
