# Country House Control

Дата старта контроля: 2026-06-08

Назначение: рабочий контрольный слой по дому, даче, воде, санузлу, подключению к сетям, расходам и бытовым блокерам.

Это личный операционный проект. Его задача - не "идеальный дом", а снятие бытовых блокеров, которые мешают жить и работать.

## Текущий фокус

1. Санузел / вода: довести времянку до рабочего состояния.
2. Колодец / точка подключения: врезка уже выполнена; осталось подключить ее к входной трубе в дом, установить счетчик и подготовить узел к пломбировке.
3. Подключение воды по процедуре ЖСК / МУП КБУ: счетчик, пломбировка, акт, договор.
4. Расходы и возвраты: чеки, лишние фитинги, список недостающего.

## Артефакты

- `current-plan.md` - текущий план работ и статусы.
- `water-connection-checklist.md` - процедура подключения воды по памятке ЖСК.
- `source-data.md` - карта исходных данных из `../repo`.

## Источник данных

Основной каталог с материалами по дому:

- `<repo-root>/life/home-family-rest/country-house/repo`

В нем лежат рабочий проект, дизайн-проект, чертежи, электрика, сметы, Blender-модели и рендеры. Контрольные документы в этом каталоге (`control/`) являются управленческим слоем поверх этих исходников.

## Правила контроля

- Домовые задачи попадают в daily plan как "личный блокер дня".
- Если задача требует внешнего человека, фиксировать owner/contact и следующий звонок/сообщение.
- Если задача требует покупки, фиксировать список: что купить, что уже есть, что можно вернуть.
- Не смешивать "сделать времянку рабочей" и "построить идеальный санузел"; это разные уровни задачи.
