# People

Люди, контакты, отношения, follow-ups.

## Folders

- `contacts/` - контактные карточки и роли.
- `relationship-notes/` - контекст отношений и история взаимодействий.
- `follow-ups/` - обещания, ожидания и следующие касания.

## Rule

Не хранить чувствительные персональные детали без практической необходимости.
