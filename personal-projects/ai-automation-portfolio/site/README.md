# Vladimir Rezin AI Automation Portfolio

Статический лендинг и портфолио личного бренда Владимира Резина для фриланс-площадок, прямых продаж и откликов на AI/automation проекты.

## Файлы

- `index.html` - основной лендинг на русском языке.
- `portfolio.html` - отдельная страница портфолио на русском языке.
- `index-en.html` - английская версия лендинга.
- `portfolio-en.html` - английская версия портфолио.
- `styles.css` - общий стиль сайта.
- `portfolio-source.md` - исходный подробный текст портфолио, из которого собраны карточки.
- `assets/` - изображения для первого экрана.

## Позиционирование

Это не отдельный бренд студии, а личный бренд:

> Владимир Резин - AI-интегратор / архитектор автоматизации для бизнеса.

Фокус:

- AI-аудит бизнес-процессов;
- AI-ассистенты по документам и базе знаний;
- AI для продаж, поддержки, HR и управления;
- CRM/API/Telegram/n8n/Make-интеграции;
- внедрение AI-assisted development без хаоса в коде;
- проверка AI-идей, ТЗ и MVP до дорогой разработки.

## Как открыть

Открыть `index.html` в браузере. Сборка и dev-сервер не требуются.
