## Optional Starting Resource

В AI-enterprise resume указан доступный стартовый ресурс:

- 2 Middle Python Developers;
- 1 Software Engineer with international experience;
- 1 Staff Engineer / Architect with international experience;
- 1 Data Scientist.

Use carefully:

- хорошо для consulting / build-team предложения;
- не нужно для классического employee CV;
- для enterprise employee vacancy по умолчанию не использовать.

## Canonical Decisions

Эти решения закрывают разрывы между старыми версиями резюме.

| Topic | Canonical CV Decision | Notes |
| --- | --- | --- |
| Arameem spelling | `Arameem / ToYou (toyou.io)` | `Arammeem` считать старым/ошибочным написанием. |
| Arameem dates | октябрь 2018 - август 2025 | Старые версии с февралём 2025 не использовать без отдельной причины. |
| Arameem scale | 100k+ delivery orders/day, peak 130k/day; 5+ financial orders per delivery order; 1M+ financial transactions/day | 150k delivery orders/day не использовать по умолчанию. |
| DAU vs users | 3k -> 300k DAU | 5m users исключить из публичных CV; оставить только для интервью после ручного подтверждения контекста. |
| Quality metrics | unit tests: production defects -60%, QA effort -90%, TTM -20%; SonarQube: code review 2x faster | Не смешивать с “production defects -90%” в одном bullet. |
| Sobit dates | public CV: сентябрь 2014 - октябрь 2018 | Формальный юридический статус “директор до сих пор” не выносить в CV; упоминать только при необходимости в разговоре. |
| Public client names | Можно использовать в RU CV; в international CV лучше обобщать | Дом.рф / Школа семи гномов / госструктуры оставить только там, где они усиливают доверие. |
| GenAI support numbers | До 90% обращений AI; default public wording: “six-figure quarterly savings” / “сотни тысяч долларов в квартал” | Точную цифру `$200k/quarter` использовать только в targeted/private CV или разговоре. |
| MedVoice claims | Doctor support only | Не писать, что AI самостоятельно ставит диагноз или назначает лечение. |

## Default Public Policy

Рабочие решения для всех производных резюме, если отдельно не задана более агрессивная версия:

- Не использовать `150k delivery orders/day`; писать `100k+ delivery orders/day, peak 130k/day`.
- Для enterprise / fintech CV добавлять транзакционный масштаб: `5+ financial orders per delivery order` и `1M+ financial transactions/day`.
- Не использовать `5m users`; писать `3k -> 300k DAU`.
- Не использовать точную цифру `$200k/quarter` в публичном international CV; писать `six-figure quarterly savings`.
- Для RU CV можно писать “экономический эффект на уровне сотен тысяч долларов в квартал”.
- Для CV по Sobit писать активный период `сентябрь 2014 - октябрь 2018`.
- Не писать `сейчас` по Sobit в employee CV, чтобы не создавать конфликт с Arameem и текущими ролями.
- Для MedVoice всегда писать только support / draft / verification / traceability.
