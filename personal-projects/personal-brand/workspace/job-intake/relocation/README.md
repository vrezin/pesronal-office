# Relocation Cards

This section stores living country notes for relocation-aware vacancy filtering.

The goal is not a pretty travel guide. The goal is to answer quickly:

- can the role be done remotely without moving;
- if not, which country is actually viable for the family;
- what the move costs up front and per month;
- whether the company helps with relocation;
- whether self-relocation is realistic;
- whether the country fits school, safety, medicine, and education needs.

## How To Use

Before analyzing a relocation-heavy vacancy:

1. Check whether full remote work is possible.
2. If not, open the relevant country card.
3. Compare:
   - rent;
   - schools for two daughters;
   - safety;
   - healthcare;
   - education quality;
   - one-time relocation costs;
   - monthly cost delta;
   - company relocation support;
   - self-relocation alternatives.
4. Check the regional shortlist for:
   - climate fit;
   - family language/adaptation speed;
   - whether the spouse can operate comfortably without immediate local-language fluency.
5. Record the verdict as:
   - `go`
   - `maybe`
   - `no`

## Update Cadence

- Refresh active country cards at least monthly.
- Refresh again before any application where relocation is likely.
- Date-stamp every update and keep the source links in the card.

## New Country Rule

When a new country appears in a vacancy or relocation discussion:

1. Create a country card immediately.
2. Pull baseline figures from the internet before using the card:
   - rent;
   - school options for two daughters;
   - safety;
   - healthcare;
   - education;
   - company relocation norms if discoverable.
3. Record the source URLs and the retrieval date in the card.
4. Mark unknown items as `TBD` rather than guessing.
5. Use the card as the relocation filter for future vacancies in that country.
