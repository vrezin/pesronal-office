# Targeted HH Resume Packages

This folder stores vacancy-specific HH resume generation packages.

The package is not just a CV text. It should describe:

- keyword gap;
- profile-level experience blocks to add or reuse;
- HH resume assembly choices;
- skills to include;
- truth check;
- copy-paste-ready HH fields.

Use this folder for high-priority vacancies where the base resume is substantively strong but may score below target because exact terms are missing.

Process:

1. Read the JD archive and analysis.
2. Follow `wiki/playbooks/hh-targeted-resume-block-library.md`.
3. Update `resume-targets/hh-experience-block-library.md` if new reusable blocks are needed.
4. Save the vacancy-specific package here.
5. Only update HH after explicit user confirmation.
