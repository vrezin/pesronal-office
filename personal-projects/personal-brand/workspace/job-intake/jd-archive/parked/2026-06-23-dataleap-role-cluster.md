## Company

Dataleap raised USD 3M from YC and angels including Aravind Srinivas and Rahul Vohra to build an agentic operating system for the enterprise. They report thousands of agents deployed across startups and Fortune 500 companies, PMF, fast revenue growth, and more enterprise inbound than they can handle.

## Reviewed Roles

### Founding Deployment Strategist - Munich

- Compensation: EUR 90K-150K + 0.10%-0.50% equity
- Location: Munich
- Key hard filters: native German, based in Munich or ready to move.
- Shape: enterprise AI transformation / rollout / use-case discovery / org training / agent adoption / C-suite / build European rollout team.

### Founding Forward Deployed Engineer - Munich

- Compensation: USD 80K-150K + 0.10%-0.50% equity
- Location: Munich
- Key hard filters: native German, based in Munich or ready to move.
- Shape: embed with DACH customers, workshops, hackathons, implementation sprints, build agents, MCP servers, integrations, SAP/internal APIs/legacy tooling, fix product limitations directly.

### Founding Engineer - San Francisco

- Compensation: USD 120K-250K + 0.20%-1.00% equity
- Location: San Francisco
- Visa: will sponsor
- Shape: CTO-level founding engineer, backend-heavy Go, full-stack/product/AI engineer, agentic systems, context engineering, tool use, coding agents, PR review, product ownership.

## Cluster Takeaway

Dataleap is one of the strongest market-category signals found so far, but the open roles split into:

- Germany-native enterprise rollout/FDE roles;
- San Francisco founding engineer role.

No role is a clean remote senior AI implementation / enterprise transformation fit from current location.
