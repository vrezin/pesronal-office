# Work

Фактическая личная работа и занятость, если появится отдельный employment-контур.

## Boundary

Поиск работы, вакансии, HH, CV, cover letters, карьерное позиционирование и personal brand не являются source-of-truth в этом разделе.

Канонический контур:

- `../personal-projects/personal-brand/`
- `../personal-projects/personal-brand/workspace/job-intake/`
- `../personal-projects/personal-brand/workspace/final-cv/`

## Folders

- `employment-search/` - tombstone/redirect на `../personal-projects/personal-brand/`; не создавать здесь новые вакансии или CV.
- `current-role/` - текущая личная работа, если появится.
- `contracts/` - договоры и условия.

## Current Contract Index

- `contracts/setronica/` - Setronica CSA/SOW contract documents.

## Rule

Если вход касается поиска работы, вакансии, резюме, сопроводительного письма или карьерного позиционирования, маршрут должен идти через `wiki/maps/personal-brand.md` и skill `personal-brand-routing`.
